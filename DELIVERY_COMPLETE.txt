HOW TO USE (BASIC WORKFLOW)
===========================

1. CREATE ARRAY
   - Click "Array" button
   - Enter size (1-20)
   - Click "Create Array"

2. ADD VALUES
   - Click any cell
   - Type value in prompt
   - Click OK

3. HIGHLIGHT CELLS
   - Right-click cell
   - Choose color
   - Cell changes color

4. ADD POINTERS
   - Click "Pointer" button
   - Select pointer name (i, j, etc.)
   - Enter index (0-20)
   - Click "Add Pointer"
   - Arrow appears above cell

5. ADD TEXT
   - Click "Text" button
   - Enter text
   - Set font size
   - Click "Add Text"
   - Drag text to position

6. TEACHING
   - Narrate each step
   - Move pointers as you explain
   - Highlight cells being discussed
   - Use text for algorithm steps

7. RESET
   - Click "Clear All"
   - Confirm deletion
   - Canvas cleared
