## ğŸ’¡ Pro Tips

1. **Establish Color Convention**
   - Red = Comparison
   - Green = Valid/Sorted
   - Yellow = Current position
   - Teal = Range/Boundary

2. **Use Clear Naming**
   - Always use 'i' for outer loop
   - Always use 'j' for inner loop
   - Matches student code expectations

3. **Narrate Actions**
   - Say what you're doing while doing it
   - "Moving pointer i to next element"
   - Helps student understanding

4. **Combine Tools**
   - Use pointers + highlighting together
   - Add text for complex explanations
   - Keep multiple elements clear

5. **Screen Sharing**
   - Works great with Zoom/Teams/Google Meet
   - 1920x1080 resolution recommended
